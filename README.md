# Gestor y Lector de Documentos PDF (Desktop App)

## Sobre el Proyecto
Aplicación de escritorio nativa desarrollada para la visualización y gestión eficiente de documentos PDF. Este proyecto nace de la necesidad de optimizar el manejo de archivos mediante una interfaz gráfica limpia y fluida, aplicando principios de Programación Orientada a Objetos (POO) y estructuración de datos en entornos locales.

Esta herramienta sirve como base tecnológica para futuras integraciones de automatización de tareas y procesamiento de lenguaje natural (NLP) / Inteligencia Artificial sobre documentos de ingeniería y flujos de trabajo de gabinete.

## Tecnologías Utilizadas

* **Lenguaje Core:** Python
* **Interfaz Gráfica (GUI):** PyQt6
* **Procesamiento de Archivos:** Librerías nativas y de manejo de PDFs en Python.

## Características Principales

* **Visualización Optimizada:** Renderizado rápido y eficiente de documentos PDF de múltiples páginas.
* **Interfaz de Usuario (UI) Intuitiva:** Diseño limpio y controles de navegación estructurados mediante PyQt6.
* **Persistencia de Estados:** Capacidad del sistema para recordar configuraciones y estados de la sesión anterior.
* **Gestión Visual:** Integración de galerías visuales para la previsualización y manejo ágil de los documentos.
